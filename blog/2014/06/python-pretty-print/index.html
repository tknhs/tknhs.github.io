Python Tips: Prettyprint

2014-06-18 | Comments

オブジェクトを整形して見やすい形で出力してくれる標準ライブラリ pprint．

短いリストにこれを使用した場合は print したときと同じように出力される．

長いリストの場合はちゃんと整形される．

なんかそれっぽい二次元配列とかも非常に見やすくなる．

リストだけじゃなくて辞書もいける．

pprint に indent, width, depth など引数を与えることができるが詳しくは公式ドキュメントで．
